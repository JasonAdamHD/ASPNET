What is the difference between inline and block HTML elements

Inline elements do not start on a new line and only take up the required space for said element. Block elements always start on a new line and stretch to the full width that they have available.

Explain why the alt attribute of the img tag is important

It is important because if the image fails to load then there will be nothing there to inform the user that there was supposed to be an image there, they are also helpful for people who use screen readers.

Explain how a using(){ } block functions and what types of objects it applies to

using applies to disposable objects so that when the object goes out of scope, the proper dispose method gets called. An example of using would be when dealing with SQL Connections

I often drone on about HTML being a semantic language. What do I mean, and how does this affect the way HTML should be written

What is meant is that the tags used should have meaning, such as using a strong tag instead of a bold tag. This affects the way we write HTML because we should use the tags the way they were designed to be used, rather than just using what makes the webpage look nice.

A business object can be used as the model as part of the MVC pattern. Why would we create a model separate from this for use with the view?

We would use a different model separate from the view so that the code responsible for the view only knows about the code that is required for creating the view.
